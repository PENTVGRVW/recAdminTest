CSSの課題
========================

CSSの課題です。

仮に作成した管理画面のHTMLとしてindex.htmlが用意してあります。
こちらにCSS、JSを追加して管理画面として完成させてください。

要件
------------------------

- レスポンシブ対応（ブレイクポイントは任意）
- `npm run build` を行なった後にindex.htmlを開いた状態で正常に表示されている
- html要素の追加を行う（class名など）
- html要素の順番の変更も可能


提出方法
------------------------

こちらのリポジトリからフォークしてプルリクエストを出してください。
